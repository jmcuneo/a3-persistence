Course Logging System

A simple system to log all the courses you've ever taken. Ever. The trick? There is no user access, so you better hope this is only being used by one person.

To create a new course, input the information in the log form, and click log. That same course can be updated by accessing it from the course ID. Remove a course in the remove form.
